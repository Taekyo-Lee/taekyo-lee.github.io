## 시작하며

드디어 블로그를 열었습니다.

이 블로그는 제가 **AI Agent 엔지니어**로 일하면서 만나는 것들을 기록하는 공간입니다. 특히 Claude Code, 그리고 제가 만들고 있는 `claude-code-fork` 프로젝트에 대한 이야기를 많이 쓸 예정입니다.

## 왜 지금 시작하나요?

몇 달 전 사내에서 **Claude Code 101** 이라는 세미나를 했습니다. 슬라이드, 실시간 시연, 그리고 GitHub에 공유한 HTML 보조자료 — 세 가지를 조합한 형태였는데, 참석자들이 *"이 조합이 너무 좋았다"* 고 피드백을 주셨어요.

그 자료들을 더 많은 사람들과 나누고 싶다는 생각이 들었습니다. 세미나가 끝나면 사라지는 지식을, 글이라는 형태로 **지속 가능한 것**으로 만들고 싶었어요.

## 앞으로의 계획

이 블로그에서 다룰 주제들:

- Claude Code 내부 구조와 커스터마이징
- `claude-code-fork` — 기업 환경에서 AI Agent를 쓰기 위한 on-premise fork
- AI Agent 시스템 아키텍처
- Prompt / Context / Harness engineering

주 1회 정도 꾸준히 쓰는 것을 목표로 하고 있습니다.
